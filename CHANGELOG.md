# Changelog

All notable changes to this project will be documented in this file. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres to
[Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-01

### Added

- **Safety boundaries:** Added a comprehensive `docs/safety_boundaries.md` document
  outlining the tool’s scope and rules of engagement. This document describes the
  default `SAFE_MODE`, permissible actions, and expectations for responsible use.
- **DISCLAIMER:** Included a `DISCLAIMER.md` clarifying that the project is intended for
  authorized red‑team activities only and does not provide stealth or bypass
  guidance beyond general security research.
- **Repository hygiene:** Introduced initial supporting files (LICENSE, SECURITY.md,
  CODE_OF_CONDUCT.md, CONTRIBUTING.md, CODEOWNERS) to meet open‑source
  best practices and provide guidance on contributing responsibly.
- **Initial release:** Tagged the first pre‑release version (`v0.1.0`) to mark the
  project’s readiness for early adopters and to enable semantic versioning going
  forward.
